## Comandos de Confirmación

Después de recibir una propuesta de cambio, responde con:

```
si          # Aplica los cambios propuestos
confirmar   # Confirma la aplicación
aplicar     # Aplica los cambios
no          # Cancela los cambios
cancelar    # Cancela la operación
```

## Comandos de Deshacer

```
deshacer    # Deshace el último cambio
undo        # Same as above
```

## Comandos de Configuración

```
config      # Abre el panel de configuración
configurar  # Same as above
settings    # Abre configuración
```
